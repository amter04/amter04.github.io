Killer Sudoku - Solveur et Présentation Web

Projet de fin d'année (1re année)

Objectif :

Développement d'un solveur de Killer Sudoku en Python, utilisant des algorithmes de recherche et des contraintes logiques pour résoudre des grilles complexes.

Technologies Utilisées :

  • Back-end : Python (pour l'algorithme de résolution).
  • Front-end : HTML, CSS et JavaScript (pour la présentation et la visualisation).

Méthodologie :

Travail en équipe avec répartition des tâches et documentation complète du projet.

PRIMOD - App Mobile E-commerce En cours

Projet Personnel - Développement Mobile

Objectif :

Création d'une application mobile fluide (iOS/Android) pour une boutique de mode urbaine. L'application gère un catalogue dynamique, des filtres par catégories (Homme/Femme/Enfant) et un panier d'achat synchronisé.

Technologies Utilisées :

  • React Native (Expo) : Pour une interface native et performante.
  • Supabase (PostgreSQL) : Gestion de la base de données (Produits, Stocks, Tailles).
  • React Navigation : Navigation fluide par onglets et piles.

Fonctionnalités clés :

Affichage des collections, page détail produit avec gestion des stocks en temps réel, panier persistant.

Gestionnaire Magasin / Site Web En cours

Projet Personnel - Développement Web

Objectif :

Développement de l'interface web connectée à la même infrastructure que l'application mobile. Ce projet sert de vitrine web et permet de tester la logique métier côté client.

Technologies Utilisées :

  • JavaScript (ES6+) : Logique front-end sans framework lourd.
  • Supabase JS Client : Connexion API et requêtes asynchrones.
  • HTML5 / CSS3 : Design responsive et moderne.

État d'avancement :

Intégration du catalogue et connexion à la base de données opérationnelles. Système de panier et tunnel de commande en développement.
