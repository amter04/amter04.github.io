PRIMOD - App Mobile E-commerce En cours

Projet Personnel - Développement Mobile

Origine du projet :

Inspiré par mon job étudiant dans une boutique de vêtements de plage, j'ai souhaité développer une solution numérique complète répondant aux besoins réels de ce type de commerce (gestion des stocks, fluidité des ventes).

Objectif :

Création d'une application mobile fluide (iOS/Android) gérant un catalogue dynamique, des filtres par catégories et un panier d'achat synchronisé.

Technologies Utilisées :

  • React Native (Expo) : Pour une interface native et performante.
  • Supabase (PostgreSQL) : Gestion de la base de données (Produits, Stocks, Tailles).
  • React Navigation : Navigation fluide par onglets et piles.

Fonctionnalités clés :

Affichage des collections, page détail produit avec gestion des stocks en temps réel, panier persistant.

Eco-Sim : Coastal Impact Tracker

Challenge Technique - Fullstack & Data Viz

Objectif :

Développement en "Hackathon mode" (une soirée) d'un tableau de bord géographique interactif. Le but est de visualiser l'impact de la montée des eaux (scénarios GIEC) sur les grandes villes côtières mondiales.

Technologies Utilisées :

  • JavaScript (ES6) : Algorithme de décision ("IA symbolique") pour évaluer les risques en temps réel.
  • Supabase : Base de données SQL pour stocker les altitudes, populations et PIB.
  • Leaflet.js : Cartographie interactive dynamique.

Fonctionnalités clés :

Simulation fluide (+0m à +5m), calcul automatique des pertes économiques (PIB impacté), navigation par continent avec zoom automatique sur la carte.

POLITICA - Narrative Intelligence Dashboard Nouveau

Hackathon 48h - IA & Big Data

Objectif :

Création d'un outil de veille géopolitique capable d'analyser en temps réel la couverture médiatique mondiale (via GDELT Project) pour détecter les biais et la polarisation des narratifs.

Technologies Utilisées :

  • Python (Flask) : Backend et moteur d'analyse.
  • Hugging Face Transformers : Modèle NLP (DistilBERT Multilingual) pour l'analyse de sentiment.
  • Supabase : Historique des recherches.
  • Chart.js : Visualisation des données (Sentiments, Sources).

Fonctionnalités clés :

Design "Premium Magazine" (The Atlanticist), classification automatique des articles (Positif/Négatif/Neutre), détection intelligente des catégories (Conflit, Économie...) et imagerie contextuelle dynamique.
